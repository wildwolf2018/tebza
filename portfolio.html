Projects

1. Integer Madness

The project is an HTML 5 2D game that combines integer arithmetic with the classic game of Snakes and Ladders. A great way to test your basic math skills while having fun at the same time. It's available on itch.io

2. Mzansi Stokvel

The project is an Etherium Blockchain DEFI app where members of a club contribute fixed sums on a fortnightly basis to a common pool. Each member receives a large sum on a rotational basis. The smart contract was written using Solidity. Source code is available on github

3. Shape Collector

A single level game where you collect as many shapes as you can while trying to beat the clock. A fun simple casual 3D game. The game was developed using C++, OpenGL and Visual Studio 2015. Source code is available on github

4. Urban Hunter

2D shooter about a bounty hunter chasing a crime boss. Created using C#, Unity and Blender 3D. Source code available on github
